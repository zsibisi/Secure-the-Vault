  • 2. Assign the message with the combination of the vault codes.

  • 4. Create forth variable to store the result for combination total
  • 5. Assign each variable with a value that matches the corresponding code in the combination, using a unique arithmetic operator for each.
  • 6. Create forth variable to store the result for combination total.
